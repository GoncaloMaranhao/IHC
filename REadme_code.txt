Para correr o projeto é preciso o android studio.
Basta abrir, no anroid studio, a pasta IHC e correr o projeto, tendo um emulador selecionado.
É na pasta IHC que se encontra o código desenvolvido para a app. 
Testado no Ubuntu, com o emulador Pixel3A e em vários telemóveis com android v9,v10,v12.
